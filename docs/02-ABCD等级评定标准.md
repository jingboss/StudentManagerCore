# ABCD 等级评定标准

## 公式

```
综合等级 = totalScore / presentFullScore = 各科得分之和 ÷ 各科满分之和
各科等级 = scoreValue / fullScore
```

## 阈值

| 等级 | 小学 | 初中 |
|------|------|------|
| A | ≥ 90% | ≥ 85% |
| B | ≥ 80% | ≥ 75% |
| C | ≥ 60% | ≥ 60% |
| D | < 60% | < 60% |

## 为什么初中阈值不同

初中科目多（9科），每科仅占约11%权重。单科极差会被其他科目稀释，适当降低阈值平衡。

例：初中生9科，1科考42%，其他满分 → 总分93.5% → A。单科失误不影响综合评级。

## 方法对比

| 方法 | 公式 | 评价 |
|------|------|------|
| **总分百分比** ✅ | 总分 ÷ 总满分 | 权重由满分值决定，与中考/高考一致 |
| 各科百分比均值 | 每科得分÷满分求平均 | 每科权重相等，抹杀满分差异 |
| 平均分 | 总分 ÷ 科目数 | 无满分基准，无法判定等级 |

## 原则

1. 总分百分比法——满分天然反映科目权重
2. 单科不否决——单科不及格只拉低均分
3. 学段区分——小学/初中不同阈值
4. 各科等级独立——单科阈值统一 A≥90% B≥80% C≥60% D<60%

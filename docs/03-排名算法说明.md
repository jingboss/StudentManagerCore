# 排名算法说明

## 规则

```
1. 总分不同 → 总分高的排前面
2. 总分相同 → 按 Subject.SortOrder 逐科比较
3. 全部相同 → 同排名（并列）
```

## 默认科目顺序

| 顺序 | 科目 |
|------|------|
| 1 | 语文 |
| 2 | 数学 |
| 3 | 英语 |
| 4 | 物理 |
| 5 | 化学 |
| 6 | 道德与法治 |
| 7 | 历史 |
| 8 | 地理 |
| 9 | 生物 |

> 使用数据库 Subject.SortOrder，学校可自行调整。

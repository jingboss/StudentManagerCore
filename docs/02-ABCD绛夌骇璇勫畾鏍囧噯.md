# ABCD 等级评定标准

## 评定方法：总分百分比法

### 公式

```
综合等级 = totalScore / presentFullScore
         = 各科得分之和 ÷ 各科满分之和（排除缺考科目）
```

各科等级（单科 badge）= `scoreValue / fullScore`

### 阈值

| 等级 | 小学 | 初中 |
|------|------|------|
| A | ≥ 90% | ≥ 85% |
| B | ≥ 80% | ≥ 75% |
| C | ≥ 60% | ≥ 60% |
| D | < 60% | < 60% |

### 为什么初中阈值更低？

初中科目多（一般9科），每科占总分的权重更小（约11%/科 vs 小学约33%/科）。当单科分数极低时，被其他科目"稀释"更严重。适当降低阈值平衡这种稀释效应。

**示例：**
> 初中生9科，1科考42%（5/12），其他8科满分 → 总分百分比 = (5+8×12)÷(12+8×12) = 101/108 = 93.5%

这个学生确实整体很强，单科失误不影响综合评级。

## 为什么不用其他方法？

| 方法 | 公式 | 评价 |
|------|------|------|
| **总分百分比** ✅ | `总分 ÷ 总满分` | **最合理。** 权重天然由满分值决定，与中考/高考总分录取逻辑一致 |
| 各科百分比均值 | 每科`得分÷满分`求平均 | 每科权重相等（1/n），抹杀了满分差异 |
| 平均分 | `总分 ÷ 科目数` | 没有满分基准，无法判定等级 |

## 核心原则

1. **总分百分比法**——各科满分已天然反映科目权重
2. **单科不否决**——单科不及格只拉低均分，不一票否决综合等级
3. **学段区分**——小学/初中使用不同阈值
4. **各科等级独立**——单科等级阈值统一 A≥90% B≥80% C≥60% D<60%

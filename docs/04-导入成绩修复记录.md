# 导入成绩 404/500 修复

## 问题

上传Excel成功，点击"确认导入"时出现 404 和 500。

## 原因

1. **404**：`UseStatusCodePagesWithReExecute("/Home/Error")` 重执行到 /Home/Error，但 Error 方法对 AJAX 未返回 JSON
2. **500**：SaveImport 无 try-catch，异常被全局处理器捕获，返回 HTML 而非 JSON
3. **LINQ 错误**：使用了 `[NotMapped]` 属性（CurrentGradeName/DisplayName），EF Core 无法翻译

## 修复

1. HomeController.Error 增强：AJAX 返回 Json，页面返回 View
2. SaveImport 添加 try-catch，异常返回 JSON
3. 预加载 ClassInfo 到内存 + 字符串字典查找，避免 LINQ 中使用 NotMapped

## 文件

- `Controllers/HomeController.cs` - Error Action
- `Controllers/ScoreController.cs` - SaveImport try-catch + 查询优化

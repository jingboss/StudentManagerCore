## 1. ABCD 等级评定标准优化 (2026-07-07)

### 修改内容
- **综合等级**：总分百分比法（总分 ÷ 总满分）
- **学段区分阈值**：

| 等级 | 小学 | 初中 |
|------|------|------|
| A | ≥ 90% | ≥ 85% |
| B | ≥ 80% | ≥ 75% |
| C | ≥ 60% | ≥ 60% |
| D | < 60% | < 60% |

- **各科等级**：单科得分率，阈值统一（不分学段）
- **核心理念**：单科不及格只拉低均分，不一票否决

### 涉及文件
- `Controllers/ScoreController.cs` - GetViewData（学生明细）、GetReportCardData（成绩报告单）

### 详细文档
见 [docs/grading-standard.md](grading-standard.md)

## 2. 排名算法优化 (2026-07-07)

### 修改内容
- 排名规则改为：**总分不同→总分高的排前面；总分相同→按科目 SortOrder 顺序逐科比较**
- 默认科目顺序：语文→数学→英语→物理→化学→道法→历史→地理→生物
- 所有比较科目分数都相同 → 同排名（并列）
- 使用数据库 `Subject.SortOrder` 顺序，不硬编码科目名

### 涉及文件
- `Controllers/ScoreController.cs` - GetViewData、GetComparisonData、GetTopStudents

### 详细文档
见 [docs/ranking-logic.md](ranking-logic.md)

## 6. 导入成绩 404/500 错误修复 (2026-07-07)

### 修复内容
- **404 错误**：`UseStatusCodePagesWithReExecute("/Home/Error", ...)` 重执行到 `/Home/Error`，但 HomeController 的 Error 方法对 AJAX 没有返回 JSON。已增强为 AJAX 请求返回 `{success:false, message:"..."}`
- **500 错误**：SaveImport 方法没有 try-catch，异常传播到全局异常处理器。已添加 try-catch 返回 JSON 错误信息
- **LINQ 翻译错误**：查询中使用了 NotMapped 属性 `CurrentGradeName`/`DisplayName`，EF Core 无法翻译。改用预加载所有 ClassInfo 到内存 + 字符串字典查找

### 涉及文件
- `Controllers/HomeController.cs` - Error 方法增强
- `Controllers/ScoreController.cs` - SaveImport try-catch + 优化查询

### 详细文档
见 [docs/import-save-fix.md](import-save-fix.md)

## 7. 空闲超时 Bug 修复 (2026-07-07)

### 修复内容
- 鼠标移动等事件监听器在超时警告弹窗显示时仍会调用 `resetIdleTimer()`，清除倒计时并重启计时器，导致弹窗永不消失
- 修复：在 `resetIdleTimer()` 顶部添加 `if (modalShown) return;`
- "继续会话"按钮有独立的重置逻辑

### 涉及文件

# 排名算法说明

## 排名规则

```
1. 总分不同 → 总分高的排前面
2. 总分相同 → 按数据库 Subject.SortOrder 顺序逐科比较
3. 所有比较科目分数都相同 → 同排名（并列）
```

## 科目比较顺序

使用数据库 `Subject.SortOrder` 字段定义的顺序，不硬编码科目名。标准配置为：

| 顺序 | 科目 |
|------|------|
| 1 | 语文 |
| 2 | 数学 |
| 3 | 英语 |
| 4 | 物理 |
| 5 | 化学 |
| 6 | 道德与法治 |
| 7 | 历史 |
| 8 | 地理 |
| 9 | 生物 |

不同学校可以自行调整 `SortOrder` 来改变比较优先级。

## 适用范围

| 位置 | 方法 | 影响 |
|------|------|------|
| 成绩分析系统 | `GetViewData` | 年级排名、班级排名 |
| 成绩对比 | `GetComparisonData` | 进退分对比中的名次升降 |
| 年级排名前10 | `GetTopStudents` | Top 10 排名 |
| 学生详情 | `StudentController.Details` | 个人排名展示 |

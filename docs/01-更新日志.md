# 更新日志

## 1. ABCD 等级评定标准优化

- **综合等级**：总分百分比法（总分 ÷ 总满分）
- **学段区分阈值**：

| 等级 | 小学 | 初中 |
|------|------|------|
| A | ≥ 90% | ≥ 85% |
| B | ≥ 80% | ≥ 75% |
| C | ≥ 60% | ≥ 60% |
| D | < 60% | < 60% |

- **各科等级**：单科得分率，阈值统一（不分学段）
- **核心理念**：单科不及格只拉低均分，不一票否决
- 文件：`Controllers/ScoreController.cs`

## 2. 排名算法优化

- 总分不同→总分高的排前面；总分相同→按 SortOrder 逐科比较
- 默认顺序：语文→数学→英语→物理→化学→道法→历史→地理→生物
- 文件：`Controllers/ScoreController.cs`

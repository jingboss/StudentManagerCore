- **分层架构**：采用经典 MVC 模式，`Controllers/` 处理 HTTP 请求，`Models/` 定义实体与视图模型，`Views/` 负责 Razor 页面渲染。
- **数据访问层**：使用 `Data/AppDbContext.cs` 作为 EF Core 上下文，通过 `Pomelo.EntityFrameworkCore.MySql` 驱动连接 MySQL 数据库，并在 `Migrations/` 中维护版本化迁移脚本。
- **服务层抽象**：`Services/PasswordHelper.cs` 提供密码哈希工具；业务逻辑（如成绩计算）通常封装在 Service 层供控制器复用。
- **中间件与安全**：自定义 `Middleware/IpRestrictionMiddleware.cs` 实现 IP 白名单访问控制；`Program.cs` 配置了 Cookie 认证、会话管理及全局异常处理。
- **独立迁移工具**：`DataMigrator/` 是一个独立的控制台项目，用于从 SQL Server 向 MySQL 进行全量数据迁移，包含字段适配与批量插入逻辑。

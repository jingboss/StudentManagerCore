# 成绩分析系统

## 概述

成绩分析系统提供多维度成绩数据分析，包含**综合分析**和**学生明细**两大核心功能模块，支持按年级、班级、科目灵活筛选。

---

## 选项卡功能

| 选项卡 | 功能 |
|--------|------|
| **综合统计** | 各科统计分析（含分数段分布）、等级分布概览 |
| **学生明细** | 每位学生的详细成绩单，含各科分数、总分、排名、**ABCD等级** |
| **成绩对比** | 班级间/年级间成绩横向对比 |
| **年级排名前10名** | 各年级/班级排名靠前的学生名单 |
| **AI成绩分析** | 基于大模型的学生/班级/科目深度分析报告 |

## 等级系统（ABCD）

### 评价标准

采用**相对比例评价法**，以单个班级全部参考学生总数为基数，按总分由高至低排序后，按预设百分比区间划分等级。

### 等级划分比例（学校统一标准）

| 等级 | 名称 | 占比 | 排名区间 | 颜色标识 |
|------|------|------|----------|----------|
| **A** | 优秀 | **25%** | 前 25% | 绿色 `#28a745` |
| **B** | 良好 | **35%** | 26% ~ 60% | 蓝色 `#007bff` |
| **C** | 及格 | **35%** | 61% ~ 95% | 黄色 `#ffc107; color:#333` |
| **D** | 不及格 | **5%** | 后 5% | 红色 `#dc3545` |

> **说明：** 等级基于**总分**在班级内的排名计算，与各科目单独统计的比率无关。

### 显示位置

- **学生明细** Tab → 学生成绩明细表 → 最后一列"等级"

## 综合统计

### 各科统计分析指标

综合统计中每个科目都有以下指标，计算方式已从传统的固定分数阈值改为**排名比例评价**：

| 指标 | 旧规则（固定阈值） | 新规则（班级排名比例） |
|------|--------------------|----------------------|
| **优秀率** | 得分 ≥ 满分 80% | 该科目分数排名前 **25%**（A档） |
| **良好率** | 70% ≤ 得分 < 80% | 排名 26% ~ **60%**（B档） |
| **及格率** | 得分 ≥ 满分 60% | A+B+C 累计 = 前 **95%** |
| **低分率** | 得分 < 满分 40% | 后 **5%**（D档） |

### 分数段分布

分数段基于**绝对分数阈值**（占满分百分比），不受排名比例影响：

| 分数段 | 含义 |
|--------|------|
| 90~100分 | 优秀区间 |
| 80~89分 | 良好区间 |
| 70~79分 | 中等区间 |
| 60~69分 | 及格区间 |
| 0~59分 | 不及格区间 |

### 其他统计指标

- **参考人数**：该科目实际参加考试人数
- **平均分**：该科目所有参考学生平均得分
- **最高分 / 最低分**：该科目得分区间
- **临界生**：得分在及格线（满分 60%）附近 ±5% 的学生，标记为帮扶对象

## 角色权限

| 角色 | 可查看范围 | 下拉框限制 |
|------|-----------|-----------|
| **管理员** | 全部年级、全部班级 | 无限制 |
| **年级级长** | 本年级所有班级 | 年级下拉锁定，不可切换 |
| **班主任** | 仅本班 | 年级+班级下拉均锁定 |

## 批量导入模板

下载成绩导入模板时，Excel 文件顶部包含：

| 行 | 内容 |
|----|------|
| 第1行 | **考试名称：** [考试名称]（14号加粗） |
| 第2行 | **创建时间：** [模板生成时间]（灰色） |
| 第3行 | 空行 |
| 第4行 | 列标题（序号、学号、姓名、年级、班级、各科目...） |
| 第5行+ | 学生数据 |

> 导入解析器自动识别表头行（查找"学号"所在行），兼容新旧两种模板格式。
